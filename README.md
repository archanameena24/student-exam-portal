# Student Exam Portal - Starter

This is a starter full-stack project:
- Backend: Spring Boot (Java 17), JPA, JWT auth
- Frontend: React (create-react-app style)

## Run backend
- Configure MySQL and update `backend/src/main/resources/application.properties`.
- From backend folder run:
  mvn clean install
  mvn spring-boot:run

## Run frontend
- From frontend folder:
  npm install
  npm start

Notes:
- This is a starter template; extend it with DTOs, validations, and production hardening.
